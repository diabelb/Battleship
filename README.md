# Battleship

Simple battleship console game.

## Requirements

PHP 8.2, Composer 2.6

## Installation

Use the package manager [composer](https://getcomposer.org/) to install foobar.

```bash
composer install
```

## Tests
Running tests
```bash
composer test
```

## Usage

To start a new game run:
```bash
php app game
```
